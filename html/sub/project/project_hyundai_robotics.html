현대로보틱스
서비스로봇

산업용 로봇에 집중하던 현대 로보틱스의 새로운 사업분야인 서빙/호텔 로봇 소개와 고객문의를 중점으로 하는
웹사이트입니다. 각 로봇에 대한 소개와 비교 기능, 각종 문의 신청 양식을 작성 기능, 홍보자료를 열람할 수 있는
기능을 사용자가 쉽게 다가갈 수 있도록 기획/디자인 되었습니다. 현대 그룹 디자인 가이드에 기반한 친근함과
세련됨이 강조되는 새로운 사업분야에 적합한 페이지 디자인을 적용하였습니다. 다양한 인터랙션과 모션을 활용
하여 사용자의 더욱 긍정적인 사이트 경험을 이끌어 냅니다. 관리자 사이트를 통해 콘텐츠 관리와 고객문의 응대가
지속적이고 통합적으로 이루어 질 수 있도록 웹사이트를 구축하였습니다.
